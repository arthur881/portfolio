Med-T

Med-T is an e-commerce website for connected clothing that can be used in the sports and medical fields.

Context :

In my first year at IIM, our year-long project was to create a complete e-commerce site. Here is mine, I designed it and coded it in php and css with javascript, it is connected to a database to manage the login and blog system. I loved making this site, and it's my first completed site that I coded from scratch.

A responsive design

A login system

Connection and registration via a form and an addition in the database. Possibility for the admin to manage the users, and the blog.
