Med-T

Med-T is an e-commerce site that specializes in connected clothing for use in sports and medical fields.

Context :

During my first year at IIM, I was tasked with creating a complete e-commerce website as a year-long project. I designed and coded the website using PHP, CSS, and JavaScript, and it is linked to a database that manages the login and blog system. I thoroughly enjoyed working on this website, and it is my first fully-developed website that I created from scratch.

A responsive design

A login system

The website features a registration and login form, and new user data is stored in the database. Admins can manage users and the blog system.
